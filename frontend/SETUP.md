# 프론트엔드 설정 가이드

## 설치 및 실행

1. **프론트엔드 디렉토리로 이동**
```bash
cd frontend
```

2. **의존성 설치**
```bash
npm install
```

3. **개발 서버 실행**
```bash
npm run dev
```

프론트엔드는 `http://localhost:3000`에서 실행됩니다.

## 백엔드 연결

백엔드 API는 `http://localhost:8601`에서 실행되어야 합니다.
프록시 설정은 `vite.config.js`에 이미 구성되어 있습니다.

## 프로덕션 빌드

```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.
