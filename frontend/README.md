# RAG Q&A Frontend

GPT 스타일의 파란색 테마를 가진 React 기반 질의응답 프론트엔드입니다.

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 환경 설정

백엔드 API는 `http://localhost:8601`에서 실행되어야 합니다.
프록시 설정은 `vite.config.js`에 있습니다.

## 기능

- 📝 질문 및 답변 인터페이스
- 📁 문서 업로드
- 📊 시스템 상태 확인
- 📄 참조 문서 표시
- 🖼️ 이미지 표시
